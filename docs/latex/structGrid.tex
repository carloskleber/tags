\subsubsection{Detailed Description}
Strutcture to represent an uniform rectangular grid to be used in specialized routines. This grid has dimensions $ L_x \, L_y $, a total of (before segmentation) $ nv = (v_x \, v_y) $ vertices and $ ne = v_y\, (v_x - 1) + v_x\, (v_y - 1) $ edges. Each edge is divided into $ N $ segments so that the total number of nodes after segmentation is $ n = v_x\,v_y + v_x\,(v_y - 1)(N_y - 1) + v_y\,(v_x - 1)(N_x - 1) $ and the total number of segments is $ m = N_x\,v_x\,(v_y - 1) + N_y\,v_y\,(v_x - 1) $ ~\newline
